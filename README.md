# 中国象棋 · Xiangqi

一个功能完整的中国象棋网页游戏，**单文件 HTML**，双击即可在浏览器中运行，无需任何依赖或构建步骤。

## 功能特性

- **标准棋盘**：9×10 棋盘（SVG 绘制），含九宫、楚河汉界、炮/兵起位点标记，红黑木纹棋子。
- **完整走子规则**：帅/将、仕/士、相/象、马、车、炮、兵/卒，含塞马腿、别象眼、炮架翻山、过河兵横走、飞将等规则。
- **胜负判定**：将军 / 将死 / 困毙判定，状态栏提示「将军！」与胜负结果。
- **中文记谱**：如「红炮二平五」「黑马八进七」，右侧实时走子记录。
- **对战模式**：
  - 双人对战（红黑轮流行棋）
  - 人机对战（可选执红 / 执黑，电脑自动应对）
- **人机 AI**：Negamax + α-β 剪枝搜索，带时间预算与着法排序；难度分简单 / 普通 / 困难（搜索深度 2/3/4）。
- **其他**：悔棋（人机模式自动回退两步）、重新开始、游戏菜单（开始游戏 / 游戏设置 / 退出）。

## 如何运行

直接用浏览器打开 `xiangqi.html` 即可。

## 玩法

1. 主菜单选择对战模式与难度，点「开始游戏」。
2. 点击棋子选中、点高亮处落子。
3. 右侧实时显示中文走子记录。
